Wideband mmWave Radio Development Platform for RFSoC Gen-3
----------------------------------------------------------------------------------
This platform combines the Otava DTRX2 Dual Transceiver mmWave Radio Card – jointly developed by Otava and Avnet – with the AMD Xilinx Zynq® UltraScale+ ™ RFSoC ZCU208 Evaluation Kit.

5G mmWave Phased Array Antenna Module Development Platform
----------------------------------------------------------------------------------
This platform enables developers to quickly create and prototype advanced 5G mmWave systems using AMD Zynq® UltraScale+™ RFSoC Gen3 and Fujikura FutureAcess™ Phased Array Antenna Module (PAAM). 

  * `Get Started <https://github.com/Avnet/Fujikura-beamforming>`_
